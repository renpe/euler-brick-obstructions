# Computational scripts for Paper 2

*Computational evidence and structural obstructions for the perfect
cuboid problem*, R. Peschmann (2026).

## Requirements

- **Python** >= 3.8 with `sympy`
- **Magma** (optional, for running the generated `.magma` scripts;
  the online calculator at https://magma.maths.usyd.edu.au/calc/ works
  for small batches)

No network access, no database credentials, no API calls. All scripts
are offline and deterministic.

## Data

`data/euler_cuboid.sqlite` bundles the relevant data for Paper 2:

| Table | Rows | Columns |
|-------|------|---------|
| `master_hits`  | 17,204  | `a, b, m, n, master_sqrt, f1` |
| `f1_factors`   | 134,309 | `a, b, m, n, prime, exponent, is_blocker` |
| `chabauty_scan`| 133,964 | `m, n, has_finite_point, points_raw` |

Large integers (some exceeding 2^63) are stored as text.

## Scripts and paper claims

| Script | Paper reference | What it does |
|--------|---------------|--------------|
| `generators/saunderson_generator.py` | Theorem 5.1 | Enumerate Master-Hits via the closed-form Saunderson formula |
| `generators/generate_chabauty_curves.py` | Section 3 (Q, R) | Emit Magma code for Chabauty0 on the genus-2 curve y^2 = Q(x)*R(x) |
| `generators/generate_himane_curves.py` | Section 5 (Himane Thm 1) | Emit Magma code for the elliptic quotient in mu = t^2 |
| `analysis/gaussian_factorization.py` | Section 4 | Classify each prime factor of f1 as Case A, B, or C |
| `analysis/classify_families.py` | Section 5 | Coverage of parametric families against the database |

## Quick examples

### Saunderson generator

```bash
python3 generators/saunderson_generator.py 10
```

Produces the first 16 Saunderson Master-Hits and verifies the Master
condition for each.

### Generate Magma code for a Chabauty0 scan

```bash
python3 generators/generate_chabauty_curves.py 50 my_scan.magma
```

Writes a Magma script covering all coprime (m, n) with m <= 50.
Run it in Magma manually:

```
> load "my_scan.magma";
```

### Gaussian factorization of a single Master-Hit

```bash
python3 analysis/gaussian_factorization.py 8 3 5 2
```

Shows the prime factorization of f1 for (a,b,m,n) = (8, 3, 5, 2),
with each prime classified as Case A, B, or C.

### Parametric family coverage

```bash
python3 analysis/classify_families.py
```

Counts how many Master-Hits in the SQLite database are produced by
Saunderson or by any of the three Himane theorems.

## Notes on Chabauty0

The Magma scripts in `examples/` and those generated by
`generate_chabauty_curves.py` assume a Magma version that supports
`Chabauty0(Jacobian(...))` for genus-2 curves over Q. Rank-0
Jacobians are verified internally by Magma; see the Magma Handbook
entry for `Chabauty0`.

## Developer script

`export_sqlite.py` rebuilds `data/euler_cuboid.sqlite` from an
internal PostgreSQL instance. It is not needed for running any of the
analyses above; the SQLite file is already included in this directory.
